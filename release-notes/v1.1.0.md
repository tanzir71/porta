# Porta 1.1.0

Porta now runs on Windows 10/11 x64 as well as Apple-silicon Macs. Both builds
bundle Cloudflare's tunnel helper, require no Porta account, and include no
analytics or telemetry.

## Downloads

- **Windows 10/11 x64:** `Porta_1.1.0_x64-setup.exe`
- **Apple silicon macOS:** `Porta_1.1.0_aarch64.dmg`
- Verify either installer with its matching `.sha256` attachment.

## Windows installation

The Windows installer is not Authenticode signed, so Microsoft Defender
SmartScreen may show **Windows protected your PC**.

1. Run `Porta_1.1.0_x64-setup.exe`.
2. If SmartScreen blocks it, select **More info**.
3. Confirm the publisher is shown as **Unknown publisher**, then select
   **Run anyway** only when the installer came from this official release and
   its checksum matches.
4. Complete the per-user installation; administrator access is not required.

## macOS installation

The macOS build is ad-hoc signed and not notarized. Open the DMG, drag Porta to
Applications, try opening it once, then use **System Settings → Privacy &
Security → Open Anyway** only after verifying the checksum.

## Highlights

- Share folders or local development ports through account-free Cloudflare
  Quick Tunnels.
- Optional password protection uses the operating system's credential store.
- Native File Explorer/Finder reveal, notification-area/menu-bar controls,
  launch at login, and automatic share restart.
- Clean shutdown prevents bundled `cloudflared` processes from being left
  behind on either platform.

Quick Tunnel URLs are public to anyone who has the link and are intended for
testing or temporary sharing, not production hosting.
