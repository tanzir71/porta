# Porta 1.0.0

Porta is a free, UI-first macOS app for sharing a folder or local web server
through a public HTTPS link. This first release bundles Cloudflare's
`cloudflared` tunnel client, requires no Porta account, and includes no analytics
or telemetry.

## Install on macOS

1. Open the DMG and drag **Porta** to **Applications**.
2. Try opening Porta once. macOS will likely block it because this release is
   ad-hoc signed and not notarized.
3. Open **System Settings → Privacy & Security** and scroll to **Security**.
4. Click **Open Anyway**, enter your login password, then confirm **Open**.

Apple makes **Open Anyway** available for about an hour after the blocked launch.
Only override Gatekeeper when the DMG came from this release and its checksum
matches. See [Apple's official instructions](https://support.apple.com/guide/mac-help/open-an-app-by-overriding-security-settings-mh40617/mac).

## Highlights

- Drag-and-drop folder sharing with public HTTPS links.
- Clean directory listings, nested navigation, downloads, media Range requests,
  and optional collision-safe uploads.
- Optional password protection backed by macOS Keychain.
- Local-port forwarding for development servers.
- Visitor, request, and bytes-served counters without telemetry.
- Menu-bar controls, launch at login, per-share auto-start, and automatic link
  copying.

## Security note

Quick Tunnel URLs are public to anyone who has the link. Enable a password for
sensitive shares, disable uploads unless needed, and stop shares when finished.
Cloudflare Quick Tunnels are intended for testing and development, have no SLA,
and are not a production hosting service.
